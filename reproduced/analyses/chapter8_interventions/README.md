# Chapter 8 Interventions

## Experimental, Outside the Verified Release Scope

This directory preserves experimental scenario-simulation code associated with
thesis Chapter 8. It is disabled by default and is not part of the maintained or
verified Chapters 4–6 public fast path or the partial Chapter 7 diagnostic. Its presence is for provenance and future
review, not a claim that the simulations are reproducible, validated, or suitable
for reuse.

Do not enable this chapter or interpret its outputs without first validating the
protected-data Chapters 4–7 workflow, reviewing the scenario assumptions against
the thesis source, and adding chapter-specific tests and release evidence.

The public verification commands (`make verify-proxy-quick`,
`make verify-proxy`, and `make verify-real`) do not run Chapter 8.
